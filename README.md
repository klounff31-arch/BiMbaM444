# 🎯 Random Task Generator

Генератор случайных задач — GUI-приложение для планирования и отслеживания задач.

---

## 📋 Описание

Приложение позволяет:
- ✅ Генерировать случайные задачи из коллекции
- ✅ Добавлять новые задачи с указанием типа
- ✅ Просматривать историю сгенерированных задач
- ✅ Фильтровать историю по типу задачи
- ✅ Сохранять коллекцию и историю в JSON-файлы

## 📦 Требования

### Версия Python
- **Python 3.6** или выше
- Рекомендуется Python 3.8+

### Зависимости
Все используемые модули входят в **стандартную библиотеку Python**:

| Модуль | Описание |
|--------|----------|
| `tkinter` | Графический интерфейс (встроен) |
| `json` | Работа с JSON файлами (встроен) |
| `os` | Работа с файловой системой (встроен) |
| `random` | Генерация случайных чисел (встроен) |
| `unittest` | Тестирование (встроен) |

> ⚠️ **Примечание:** На некоторых Linux-системах tkinter может требовать отдельной установки:
> ```bash
> # Ubuntu/Debian
> sudo apt-get install python3-tk
> 
> # Fedora
> sudo dnf install python3-tkinter
> 
> # macOS
> brew install python-tk
